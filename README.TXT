PROJECT TITLE:     Poison Airlock Escape
PURPOSE OF PROJECT: An AP Computer Science A final project: a text-based
                    escape-room adventure wrapped in a browser UI. The
                    player is trapped in a space-station airlock that is
                    slowly filling with poisonous gas, and must solve
                    puzzles, collect items, and pick between A/B choices
                    to escape by the vent, the escape pod, or the main
                    door before time runs out.
VERSION or DATE:   1.0 (April 2026)
HOW TO START THIS PROJECT:
                    Option 1 - Browser UI (recommended):
                      1. Open the project in BlueJ.
                      2. Right-click PoisonAirlockWebServer and choose
                         "void main(String[] args)".
                      3. Open http://localhost:8080 in a web browser.
                    Option 2 - Console version:
                      Right-click PoisonAirlockEscape and run main.
AUTHORS:           Gary (APCSA Final Project)
USER INSTRUCTIONS:
                    - Click hot-spots in the room or type commands into
                      the prompt: look, examine [object], use [item],
                      solve [thing], escape [main / vent / pod],
                      inventory, status, help, quit.
                    - Watch the timer - the gas will kill you in
                      600 seconds (the oxygen mask buys extra time).
                    - Critical A/B decisions pop up an overlay; your
                      picks change the story and your final score.
                    - "Reset Simulation" in the top right starts over.
